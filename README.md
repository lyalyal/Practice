Тема проєкту: Електроний читацький щоденник 
Вебзастосунок для створення особистої книжкової полички

Основні можливості:
- додавання, редагування, видалення, пошук книг
- загальна статистка по доданих книгах
- заповнення форми додавання книги
- адаптивність застосунку
  
Технічний стек:
Backend: Node.js, Nest.js, TypeScript
Frontend: React (Vite), TypeScript
Система контролю версій: Git + GitHub
Інструменти: VS Code, cURL / Postman / Bruno

Щоб запустити застосунок потрібно використати:
git bash
git clone [https://github.com/lyalyal/Practice.git]
cd Practice

В першому вікні 
cd backend
npm install
npm run start:dev

Інше вікно 
cd frontend
npm install
npm run dev
