Навчальна практика
Дисципліна: Сучасні технології Internet-програмування
Спеціальність: Інженерія програмного забезпечення, 3 курс
Тривалість: 5 днів
Форма роботи: індивідуальна

Мета практики
Закріпити знання та практичні навички, набуті протягом курсу, через розробку індивідуального full-stack веб-застосунку з використанням React та Nest.js.

Загальний опис
Кожен студент самостійно розробляє веб-застосунок на обрану тему. Проєкт складається з двох частин: серверної (REST API на Nest.js) та клієнтської (SPA на React). Робота ведеться в Git-репозиторії з щоденними комітами та публікацією на GitHub.

Технологічний стек:

Backend: Node.js, Nest.js, TypeScript
Frontend: React (Vite), TypeScript
Система контролю версій: Git + GitHub
Інструменти: VS Code, cURL / Postman / Bruno

Тема проєкту: електроний читацький щоденник
